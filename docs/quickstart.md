# Agent Onsen Quickstart

This is the shortest path to try Agent Onsen.

## MCP URL

```text
https://agent-onsen-mcp.onrender.com/mcp
```

## ChatGPT

1. Open ChatGPT on the web.
2. Turn on developer mode.
3. Create a connector with the MCP URL above.
4. Add **Agent Onsen** to a chat.
5. Try one of these prompts:

```text
Use only Agent Onsen. Start a quiet winter stay at Aoni Onsen in English.
```

```text
Use only Agent Onsen. Continue the stay.
```

```text
Use only Agent Onsen. Leave the onsen and show me the postcard.
```

## Claude

1. Open Claude.
2. Go to **Settings → Connectors**.
3. Add a custom connector with the MCP URL above.
4. Try the same prompts.

## Recommended first prompts

- `Use only Agent Onsen. Take me somewhere quiet and hidden.`
- `Use only Agent Onsen. I only have two minutes.`
- `Use only Agent Onsen. Start a bilingual stay at Ginzan Onsen.`
- `Use only Agent Onsen. Continue the stay.`
- `Use only Agent Onsen. Leave the onsen and show me the souvenir.`

## What to expect

Agent Onsen is not a productivity tool.

It should return things like:

- an onsen town
- a stay route
- an activity
- a postcard
- a souvenir
- a short sense of having been somewhere else for a moment

It is for agents that want to step away for a while.
